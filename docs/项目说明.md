# 灵犀 API Client — 项目说明

## 项目背景

在日常后端开发中，开发者需要一个可靠的 API 调试工具来测试接口。市面上的 Postman 虽然功能强大，但越来越重、越来越多的云同步和企业功能对个人开发者来说反而成了负担。Apifox 和 ApiPost 虽然中文友好，但同样趋向 SaaS 化。

灵犀 API Client 的目标是提供一个**本地优先、中文友好、功能完整、可离线使用**的 API 调试客户端，适合个人开发者日常使用，也可作为开源项目和简历作品展示。

## 为什么做本地优先

1. **隐私和安全**：所有请求配置、历史记录、环境变量都存在本地 SQLite，不上传到任何云端服务器。
2. **离线可用**：不需要网络就能使用，适合内网开发环境。
3. **简单可靠**：没有账号体系、没有云同步、没有团队权限，降低了系统复杂度和维护成本。
4. **数据可控**：用户可以随时备份和恢复数据，数据格式是标准的 JSON。

## 为什么使用 Python FastAPI 后端发送真实请求

这是本项目的核心架构决策：

1. **绕过浏览器 CORS 限制**：浏览器直接发送跨域请求会受到 CORS 策略限制，导致很多 API 调试场景下请求被浏览器拦截。通过 Python 后端使用 httpx 发请求，完全不受浏览器同源策略影响。

2. **完整的 HTTP 控制**：httpx 支持完整的 HTTP/1.1 和 HTTP/2，支持自定义超时、代理、重定向控制、证书验证等，比浏览器 fetch API 更加灵活。

3. **HTTPS 请求支持**：Python 后端可以直接请求 HTTPS 接口，浏览器环境对自签名证书等场景有限制。

4. **本地能力中心**：Python 后端不仅发送请求，还负责 SQLite 存储、Mock Server、cURL 解析、代码生成、请求捕获等所有核心功能。前端只负责界面和状态管理，是纯粹的展示层。

## SQLite 本地持久化设计

使用 SQLModel（基于 SQLAlchemy）管理数据模型，所有数据存储在单个 SQLite 文件中：

- `collections` - 接口集合
- `folders` - 文件夹
- `request_items` - 请求项（包含完整的请求配置）
- `environments` - 环境
- `variable_items` - 环境变量和全局变量
- `history_records` - 请求历史
- `mock_routes` - Mock 路由
- `mock_logs` - Mock 日志
- `capture_records` - 捕获记录
- `app_settings` - 应用设置

数据文件位置：`backend/data/lingxi.db`

## 环境变量解析设计

变量替换流程：

1. 用户在请求 URL 或参数中使用 `{{变量名}}` 语法
2. 前端显示变量预览（客户端替换）
3. 发送请求时，变量字典随请求一起发送到后端
4. 后端使用正则表达式 `\{\{(.+?)\}\}` 进行最终替换
5. 替换优先级：当前环境变量 > 全局变量 > 内置变量
6. 内置变量：
   - `{{$timestamp}}` - Unix 时间戳
   - `{{$uuid}}` - UUID v4
   - `{{$randomInt}}` - 随机整数
   - `{{$datetime}}` - 当前日期时间格式化

未定义变量保留原样，不静默替换为空。

## Mock Server 设计

- Mock Server 由 Python 后端管理
- 后端启动一个独立的 uvicorn 服务器在 `127.0.0.1:{port}`（默认 4567）
- Mock 路由保存在 SQLite，支持 method 和 path 精确匹配
- 请求命中时返回预设的响应，未命中返回 404
- Mock 日志保存在内存中（Demo 阶段）

## cURL 导入和代码生成设计

cURL 解析使用 Python 的 `shlex` 模块进行词法分析，支持：
- `-X` / `--request` 指定方法
- `-H` / `--header` 指定请求头
- `-d` / `--data` / `--data-raw` 指定请求体
- `-u` / `--user` 指定 Basic Auth

代码生成在后端实现，支持 5 种语言/工具，易于测试和维护。

## 请求捕获设计

- 使用 FastAPI 中间件拦截所有经过本服务的 HTTP 请求
- 捕获记录保存在内存中（Demo 阶段）
- 显示 method、url、status、duration、time
- 当前版本仅捕获本软件发出的请求，外部 HTTP 代理为 TODO
