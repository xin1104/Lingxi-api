## 🔴 任务一：Tests 脚本执行引擎（后端核心）

### 1.1 新建 `backend/app/features/request_workbench/test_runner.py`

实现一个 Python Tests 执行引擎，支持类 JavaScript DSL：

```python
# 支持的 DSL 语法（与前端 testRunner.ts 一致）
pm.test("测试名称", () => {
  pm.expect(value).toBe(expected)           # 严格相等
  pm.expect(value).toContain(substring)      # 字符串包含
  pm.expect(value).toHaveStatus(code)        # 状态码断言
  pm.expect(value).toHaveHeader(name, val)   # 响应头断言
  pm.expect(value).toBeJson()               # 是合法 JSON
  pm.expect(value).toBeLessThan(n)          # 小于
  pm.response.status                         # 状态码
  pm.response.headers                        # dict
  pm.response.body                           # 字符串
  pm.response.json()                         # JSON 解析
  pm.response.time                           # 响应耗时 ms
  pm.environment.get(key)                    # 环境变量
  pm.environment.set(key, value)             # 设置环境变量
})
```

**实现方案**：Python `exec()` + 受限 `globals`（`__builtins__` 设为空字典防止沙箱逃逸）。

**关键类结构**：
```python
class PmObject:
    """提供给脚本的 pm 对象"""
    def __init__(self, response, env_vars, results_list)
    def test(self, name, fn) → None          # 执行测试，捕获异常
    def expect(self, value) → ExpectChain    # 返回断言链
    # response / environment 属性

class ExpectChain:
    """链式断言"""
    def toBe(self, expected) → None
    def toContain(self, substring) → None
    def toHaveStatus(self, code) → None
    def toHaveHeader(self, name, value) → None
    def toBeJson(self) → None
    def toBeLessThan(self, n) → None

class TestRunner:
    def __init__(self, response, env_vars)
    def run(self, script: str) → list[dict]  # 返回测试结果列表
```

### 1.2 新建 `backend/tests/test_test_runner.py`

覆盖以下场景：
- 空脚本 → 无结果
- `toBe` 通过/失败
- `toContain` 通过/失败
- `toHaveStatus` 通过/失败
- `toBeJson` 通过/失败
- `toBeLessThan` 通过/失败
- 多条测试混合通过/失败
- 语法错误脚本

### 6.1 修改 `backend/app/features/capture/router.py`

将 `_captured_requests` 内存数组改为直接读写 `CaptureRecord` 表（模型已在 `models.py` 中存在）：

```python
@router.get("/capture/records")
async def get_capture_records(limit: int = 100, session: Session = Depends(get_session)):
    records = session.exec(
        select(CaptureRecord).order_by(CaptureRecord.created_at.desc()).limit(limit)
    ).all()
    return ApiResponse(data=[r.model_dump() for r in records])

@router.delete("/capture/records")
async def clear_capture_records(session: Session = Depends(get_session)):
    session.exec(delete(CaptureRecord))
    session.commit()
    return ApiResponse(message="已清空")
```
